Features

Everything you need, nothing you don't.

Built from the ground up to feel native on both desktop and mobile.

Custom Icons

Pass your own rbxassetid:// for the window title, loading screen, tabs, and notifications.

Searchable Tabs

A search bar at the top of the sidebar filters tabs in real time. Perfect for hubs with many tabs.

Smart Dropdowns

Capped at 3 visible options with built-in scrolling. No more giant lists pushing your UI off the screen.

Mobile First

Draggable floating toggle, touch-friendly sliders, and tap-vs-drag detection that actually works.

Config Saving

Flag any element with Save = true and it's automatically persisted between sessions.

Animated Loader

Gradient background, expanding logo circle, and typewriter intro text. Make a great first impression.

local Window = SarXNight:MakeWindow({
    Name = "My Hub",
    InterfaceId = "rbxassetid://11436111346",
    LoadingLogoId = "rbxassetid://11436111346",
    IntroEnabled = true,
    IntroText = "Welcome",
    SaveConfig = true,
    ConfigFolder = "MyHub"
})
Tab:AddToggle({
    Name = "Auto Farm",
    Default = false,
    Flag = "autofarm",
    Save = true,
    Callback = function(v) print(v) end
})
Tab:AddSlider({
    Name = "Speed",
    Min = 16, Max = 200, Default = 16,
    Increment = 1,
    ValueName = "studs/s",
    Flag = "speed", Save = true,
    Callback = function(v) end
})
Tab:AddDropdown({
    Name = "Mode",
    Options = {"One", "Two", "Three", "Four"},
    Default = "One",
    Flag = "mode", Save = true,
    Callback = function(v) end
})
SarXNight:MakeNotification({
    Name = "Loaded",
    Content = "Welcome to the hub!",
    Image = "rbxassetid://10709751939",
    Time = 5
})
