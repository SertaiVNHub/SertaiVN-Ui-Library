Features

Everything your script hub needs

Battle-tested elements that work flawlessly on PC and Mobile, with smooth animations and clean defaults.

Animated Loading Screen

Typewriter intro text and a custom logo with gradient backdrop. Configurable via LoadingLogoId.

Search-Filtered Tabs

Built-in search box filters your sidebar tabs in real time — perfect for hubs with 10+ tabs.

PC + Mobile Native

Sliders, dropdowns, dragging, and the floating toggle all work natively on touch screens.

Smart Dropdowns

Capped at 3 visible options with smooth internal scrolling. No more giant dropdowns crowding your UI.

Persistent Config

Tag any element with a Flag and Save = true to auto-save and restore on next session.

Custom Branding

Set your own logo with InterfaceId — the icon shows in the title bar AND mobile toggle.

Image Notifications

Sleek bottom-right notifications with custom icons. Auto-dismiss with smooth fade.

Drop-In Compatible

Legacy Icon and IntroIcon keys still work — upgrade from v2.5 in seconds.

Get Started

Up and running in 30 seconds

Drop the loader into any executor, write your first window, and you're live.

lua 
local SarXNight = loadstring(game:HttpGet("https://raw.githubusercontent.com/YOU/SarXNight/main/Source.luau"))()

local Window = SarXNight:MakeWindow({
    Name = "My First Hub",
    InterfaceId = "rbxassetid://11436111346",
    LoadingLogoId = "rbxassetid://11436111346",
    IntroText = "My First Hub",
    SaveConfig = true,
    ConfigFolder = "MyFirstHub"
})

local Main = Window:MakeTab({ Name = "Main" })

Main:AddToggle({
    Name = "Auto Farm",
    Flag = "autofarm", Save = true,
    Callback = function(v) print("AF:", v) end
})

SarXNight:Init()
